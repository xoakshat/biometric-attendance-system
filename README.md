# Biometric Attendance System

A face recognition-based attendance system built using Python and Appwrite database.

## Features

* Face detection & recognition
* Automatic attendance marking
* Cloud database integration (Appwrite)
* Simple frontend dashboard

## Tech Stack

* Python
* OpenCV (Face Recognition)
* HTML/CSS
* Appwrite (Backend Database)

## How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run backend:

```
python main.py
```

3. Open frontend:

* Open `index.html` in browser

## Notes

* Photos folder contains registered faces
* Attendance data is stored in Appwrite database
* Configure your own Appwrite project if needed
